# WEBPACK DEMO

## build bundle

```bash
npm run build

# or with --no-color option
npm run build -- no-color
```

## SIMPLE COMPILING OF BUNDLE FILE

```bash
npx webpack
# or
npx webpack --config webpack.config.js
```
